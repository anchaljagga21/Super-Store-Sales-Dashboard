# Super-Store-Sales-Dashboard
This repository showcases a dynamic **Superstore Sales Dashboard** built using **Power BI** to analyze 
sales performance, profitability, discount impact, and category-wise trends. 
The dashboard focuses on understanding customer segments, regional performance, 
and overall business health through powerful analytics and interactive visualizations.

**Project Objectives**
- Analyze overall **Sales**, **Profit**, and **Quantity Sold**
- Identify **Top & Bottom Performing Products**
- Understand **Category & Sub-category trends**
- Evaluate **Regional performance and customer segments**
- Measure **Profit Margin %**, **YoY Growth**, and **Discount Impact**
